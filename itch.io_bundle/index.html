Itch.io's Bundle for Racial Justice and Equality

A while ago Itch.io released their Bundle for Racial Justice and Equality. It got massively popular and ended up with over 1700 games/comics/RPG rules. Sorting through this content proved troublesome. This is an attempt to categorise those games.

This project used Python to access the webpage of each game, which was then scraped for more data using BeautifulSoup, a Python module for parsing HTML. The review scores, category, and a few other useful data were saved to a local file, which is also hosted here and displayed sorted by category below.
