Performance

A player recieves 10 points for a solo victory, 5 points for being part of a draw, 1 point for surviving a solo victory, and 0 points for being knocked out. This cumulative score is then divided by the total number of games the player has participated in.

Alex's Performance

Alex's Performance is an alteration of the default performance created to boost Alex's score. Players earn 0 points for a death and 1 point for a survive. For a game with N players, a solo win earns the player N points. In the case of M players drawing the game, each player earns N/M points. Each player's cumulative score is then divided by the total number of games they participated in.
