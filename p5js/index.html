p5.js Projects

p5.js is a neat little JavaScript library for animations and artwork using the HTML canvas element. It adds a lot of useful methods to make creating animations faster and easier. A lot of the language is based on Processing, a programming language to make things that look cool and move around.

Here's some stuff I've made with p5.js while screwing around:

  • Bouncing Balls
  • Orbits
  • Random Picker
  • Conway's Game of Life
